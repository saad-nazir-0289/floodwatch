# Contributing

Thanks for improving `Indus Urban FloodWatch`.

## Development basics

- Use Python 3.14 or a recent supported Python 3 version.
- Install dependencies with `pip install -r requirements.txt`.
- Run locally with `streamlit run app.py`.

## Contribution guidelines

- Keep the app focused on satellite, drone, and aerial imagery.
- Prefer lightweight CPU-friendly changes over heavyweight model dependencies.
- Keep demo outputs explainable and visually strong.
- Test imports and a local Streamlit startup before opening a pull request.

## Good contribution ideas

- improve overlays and legends
- add new demo overhead scenes
- refine flood or built-up heuristics
- improve docs and deployment guidance
- polish UI copy and usability
